A Classe Huffman.java é a classe principal de compressão.
Um arquivo de entrada chamado entrada.txt é disponibilizado dentro da pasta /Files.

Para rodar a compressão, siga os passos:
        javac *.java
        java Huffman <arquivo.txt>


Após a compressão, os arquivo normal.bin, compress.bin e tableFrequencies.txt são gerados na pasta /Files

Para rodar a descompressão, basta apenas digitar:
        java Decompress


que a palavra resutante será impressa na tela.
